GPD XD+ (Plus)
==============

The XD+ (Plus) (codenamed _"xds"_) is a Mediatek MT8176-based Android handheld console from GPD and the successor to the original Rockchip RK3288-based XD.

Basic   | Spec Sheet
-------:|:-------------------------
CPU     | 2.1GHz Quad-Core MT8176
GPU     | PowerVR GX6250
Memory  | 4GB RAM
Ships   | Android 7.0
Storage | 4GB
Battery | 6000 mAh
Display | 5" 1280 x 720 px
